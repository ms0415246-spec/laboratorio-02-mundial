# Laboratorio 02 - Mundial 2026

Aplicación web desarrollada para el curso ISW-521 que consume una API REST del Mundial 2026 utilizando JavaScript moderno (ES Modules).

## Funcionalidades

- Ruta del Campeón
- Goleadas del torneo
- Ranking defensivo (El Muro)
- Analítica de Estadios
- Radar de Empates

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES6 Modules
- Fetch API
- LocalStorage
- API REST World Cup 2026

## Estructura del proyecto

```
css/
js/
 ├── api/
 ├── services/
 ├── ui/
 ├── utils/
index.html
```

## Características implementadas

- Consumo de API mediante Fetch.
- Manejo de errores HTTP.
- Reintentos automáticos.
- Caché con LocalStorage.
- Alertas dinámicas.
- Navegación SPA.
- Diseño responsive.
- Código modular.
